La versione italiana del programma Š nella directory 'it'.

If you want to rebuild the program from its source code you
have to add to the 'src' directory the header file app_msgs.h.
Copy it either from the 'en' or the 'it' directory.
The program is currently built with Open Watcom 1.5 but I also
succesfully built it with IBM Visual Age C++ 3.5.

copiare nella directory 'src' il file header app_msgs.h dalla
directory 'it' o 'en'.
Il programma corrente Š stato compilato con Open Watcom 1.5, ma
Š stato compilato con successo anche con IBM Visual Age C++ 3.5.

To learn how to use the program read fsj.txt in the 'en' or 'it'
directory or just run fsj.exe.

Per imparare ad usare il programma si legga il fsj.txt nella
directory 'it' o 'en' o si esegua direttamente fsj.exe.
